# wst

## безлопастная чистка данных профиля стим
основных папок Steam управлением над профилями:
`appcache`,
`config`,
`depotcache`,
`dumps`,
`friends`,
`logs`.
### Перед запуском обязательно отредактируйте любым текстовым редактором, файл `wst`:
Основные папки для очистки:
`appcache`,
`config`,
`depotcache`,
`dumps`,
`friends`,
`logs`.

```PowerShell
$folders = @(
            "V:\Program Files (x86)\Steam\appcache",
            "V:\Program Files (x86)\Steam\config",
            "V:\Program Files (x86)\Steam\depotcache",
            "V:\Program Files (x86)\Steam\dumps",
            "V:\Program Files (x86)\Steam\friends",
            "V:\Program Files (x86)\Steam\logs"
```

#### Важно
- Закройте Steam перед запуском
- Убедитесь, что пути указаны верно
- Папка с `userdata` — это ваши аккаунты или подругому тут храняться настройки всех игр, настройки стим для данного пользователя, бывают и сохранения сессий.

### Примичание
- Если у вас не скачивается файл, то выберите скопировать через `raw .file`
- вставьте в текстовый редактор и выбрите в кодировке `UTF-8 c BOM`, если у вас кириллица не коректно отображается
- Сохраните как и обязательно расширение файла `.ps1`.
#### Запуск скрипта
1. Запустите скрипт `wst.ps1` правой кнопкой по нему и выполнить с помощью PowerShell
2. Вас привествует скрипт на первом этапе, показывает что находится в папке `userdata`, которую вы отметите в самом скрипте. И в дальнейшем вы решаете продолжать работу над настройками сохранений. продолжив вы увидите режимы
3. Режимы:
- `1` — очистка папок
- `2` — очистка + с userdata:
    - запрос на удаление
    - перенос профиля
- `3` — дополнительно, менять имя папки на дату

### Пример
Для примера сделал на своей основе дисков. У вас порядок может отличаться.
У меня стим установлен на диске `V:`: что видно на видео, как происходит процессе видео.
- при выборе режима 2 предлагается чистка или перенос:
    - Обязательная чистка временных файлов как в певом пункте скрипта.
    - предлагается очистить `userdata` 
    - или содержимое перенести в папку `T:\userdatawst`
